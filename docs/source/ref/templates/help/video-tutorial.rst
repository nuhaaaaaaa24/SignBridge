===================
Video Tutorial Page
===================

Overview
========

The Video Tutorial page provides users with guided instructions on how to use
the SignBridge application.

The page is designed to support onboarding and improve usability by helping
users understand the system workflow before starting a communication session.

Purpose
=======

The tutorial page acts as a user support resource and demonstrates how to:

* create an account
* login
* create a session
* join a session
* edit profile
* access help pages
* contact support
