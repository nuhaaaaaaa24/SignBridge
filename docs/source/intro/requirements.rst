============
Requirements
============

Python
======

SignBridge requires Python 3.14 or above.

Browser
=======

SignBridge is compatible with all modern browsers including:

* Chrome
* Firefox
* Safari
* Edge

.. note::

   Camera and microphone permissions must be granted in your browser to use
   the video call and sign language recognition features.

Database
========

* **Development** — SQLite 
* **Production** — PostgreSQL
